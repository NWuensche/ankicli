# Anki CLI Program
This is a small python program which helps you adding Anki cards on AnkiWeb through the command line.

## How to Start
Install PhantomJS
`pip3 install selenium`
`python3 anki_cli.py`

## How to Use it
First off, you have to add your AnkiWeb Username and Password into the program. Next off, you have to decide which deck you want to add cards to. Lastly, just add cards and press `Ctrl+c` when you're done.
